# Ideas

Ideas for future projects and tools.

- [Bonjour-Aware Reverse Proxy](bonjour-reverse-proxy.md) — A local reverse proxy that
  discovers dev servers via Bonjour/mDNS and Docker events, making them accessible via
  `<service-name>.local`.
- [Keep Git Commit Messages Updated During Claude Code Sessions](claude-code-commit-message-updates.md) —
  Get Claude Code to maintain a running commit message summary as it works, so you always
  have a persistent overview of changes in the VSCode source control UI.
